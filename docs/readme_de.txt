INSTALLATIONSANLEITUNG
----------------------
1) Kopieren Sie MediaRepository in das Verzeichnis modules.
2) Initialisieren und aktivieren Sie MediaRepository in der Modulverwaltung.
3) Setzen Sie das Verzeichnis `userdata/MediaRepository/` inklusive aller Unterordner auf beschreibbar.

Bei Fragen und anderen Anmerkungen besuchen Sie unsere Homepage http://zikula.de.
